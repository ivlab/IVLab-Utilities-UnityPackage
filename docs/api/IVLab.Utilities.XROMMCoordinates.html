    Class XROMMCoordinates

    Uses the CoordConversion class to provide routines for converting from XROMM coordinates to Unity coordinates.

    XROMM coordinates are: Right-Handed Up = +Z Forward = +Y Right = +X

    Unity coordinates are: Left-Handed Up = +Y Forward = +Z Right = +X
